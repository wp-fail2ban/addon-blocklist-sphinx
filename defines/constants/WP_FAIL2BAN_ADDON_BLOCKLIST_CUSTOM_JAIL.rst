WP_FAIL2BAN_ADDON_BLOCKLIST_CUSTOM_JAIL
---------------------------------------

.. versionadded:: 1.0.0

----

Changes the log format to match entries in ``wpf2b-blocklist-hard.conf`` instead of ``wordpress-hard.conf``.

.. code-block:: php

  define('WP_FAIL2BAN_ADDON_BLOCKLIST_CUSTOM_JAIL', true);

**Default:** :php:`false`
