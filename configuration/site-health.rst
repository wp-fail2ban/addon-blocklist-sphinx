Site Health Tool
----------------

.. versionadded:: 2.1.0

----

WP fail2ban Blocklist uses the standard WordPress Site Health tool to check that the Secret Key is available for communication with the BNS, and if possible, that ``fail2ban`` has been configured correctly.
